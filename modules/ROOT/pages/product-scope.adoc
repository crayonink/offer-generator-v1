== Summary

* *Complete (8):* VLPH, HLPH, Tundish, Blowers, Burners, HPU/PU, Recuperators.
* *Scaffolding (3):* REGEN Burners, Billet Reheating Furnace, Box Type Furnace.
* *Partial (2):* Fuel & Air Skids/Trains, Rad Heat Gas Elements.
* *Pending (18):* all remaining product lines above.

[NOTE]
====
Module acronyms: *VLPH/HLPH* = Vertical/Horizontal Ladle Preheater · *HPU* = Heating & Pumping Unit ·
*PU* = Pumping Unit only · *SNSF BRF* = Billet Reheating Furnace · *BTF* = Box Type Furnace ·
*REGEN* = Regenerative Burner · *TLC* = Torpedo Ladle Car.
====
