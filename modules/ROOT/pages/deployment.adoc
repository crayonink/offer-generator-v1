== Configuration files

[cols="1,3", options="header"]
|===
| File | Purpose
| `Procfile` | process/start command
| `nixpacks.toml` | build configuration
| `railway.toml` | Railway service config
| `config.py` | app constants (`MARKUP = 1.8`, `USD_RATE`, …)
| `requirements.txt` | Python dependencies
|===

== Database

The SQLite database `vlph.db` is *committed to the repo*.

[IMPORTANT]
====
DB edits do not reach Railway until the `.db` file itself is committed and pushed
in the same change. Always commit `vlph.db` alongside related code.
