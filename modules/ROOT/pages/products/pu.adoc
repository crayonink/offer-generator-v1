== Calculation Flow

Shared with HPU except where noted. PU posts to `POST /api/generate-pu-quote` â†’ `_generate_pumping_unit_offer(req, mode="pu")` (`main.py:3350-3356`). Both products run the _same helper, form, template shell and render path_; only the price source, model-code prefix, equipment label, filename infix and template file differ (selected by `mode`, `main.py:3115-3122`).

. *Form â†’ request.* `pu_costing.html` posts the same payload to `/api/generate-pu-quote` (`pu_costing.html:523`); same `HpuQuoteRequest`.
. *Flow-LPH lookup.* _Identical_ to HPU â€” `pumping_unit_price.flow_lph` by (kW, variant) (`main.py:3134-3139`). The pump skid is shared.
. *PU selection + unit price â€” OMITS the HPU markup step.* `unit_price = pumping_unit_price.sell_price` for (kW, variant), read directly (`main.py:3162-3171`). This price is already markup-included (rebuilt from `hpu_master` minus heater/thermostat Ã— 1.8), so `HPU_MARKUP` is NOT applied here (contrast the HPU page, step 3). Mirrors `GET /api/hpu/price?mode=pu` (`main.py:3096-3103`). Errors out if no `pumping_unit_price` row.
. *Commercial-adjustment formula â€” identical* (`main.py:3186-3192`): `_adj_total = unit_price * qty * (1 + (_pf+_des+_neg)/100) + _trn`; negotiation ADDS; round to nearest â‚¹1000 when any adjustment set; `offer_unit = _final_incl / qty`.
